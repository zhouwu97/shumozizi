# 空间几何证据图

1. 先读取当前结果、坐标系、单位和关键事件。只有空间关系会影响模型选择、可行性或结论时才画三维图；若二维距离图更容易核验，应优先用二维。
2. 使用 `src/shumozizi/geometry/visual.py` 的有限线段、轨迹、圆柱、球形区域、事件点、等比例坐标轴和统一导出图元。按需读取 `vendor/nature-skills/nature-figure/` 的多面板与图形 QA 规范，以及 `vendor/scientific-agent-skills/scientific-visualization/` 的物理尺寸、配色和矢量导出参考。
3. 一套完整空间证据通常包含可旋转理解的三维总览、XY 俯视、XZ 侧视和临界事件局部放大；按当前结论裁剪，不强制固定四图。三轴必须等比例，标明单位、对象、轨迹方向和关键时刻，避免透视拉伸制造相交错觉。
4. 脚本读取当前运行登记的真实 JSON，保存到 `code/figures/`，输出 PDF/SVG/PNG 到 `figures/` 并登记渲染收据。导入的示例资产只作布局参考，不能进入论文证据。

图形最终仍需检查对象是否真实出现、边界是否可辨和图文主张是否一致；文件成功生成不等于几何表达正确。
