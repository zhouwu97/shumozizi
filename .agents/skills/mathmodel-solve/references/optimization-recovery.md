# 非光滑优化的恢复性参考

仅在当前题属于黑箱、稀疏命中、离散判定或仿真优化时读取。它补充主 Skill 的 P0，不替代题目判定器。

令原始保守目标为 \(F(x)\)，已知可行基线为 \(x_0\)。先用同一 `score(x, mode)` 计算 `proxy`、`coarse_exact` 与 `fine`，并要求 \(F_{fine}(x_0)>0\)。每个随机 seed 的候选集必须含 \(x_0\)，最终解满足：

\[
x^*\in\{x:\text{feasible}(x),\ F_{fine}(x)\ge F_{fine}(x_0)-\varepsilon\}.
\]

代理可使用距离裕量、连续事件根定位或基线邻域采样；但报告值必须回到原始保守判定的细网格或事件复算。若非基线候选均无正 proxy，而基线为正，代理不提供邻域信息：停止该问题，换代理或局部方法，不扩展到下一问。
