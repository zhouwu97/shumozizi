# A/B 结果

## 泄漏修复与复现

旧 `q1-primary` 已因入睡方式目标泄漏被标记为 `superseded` 且禁止进入论文。修复后的
`q1-primary-leakage-fixed-v4` 与同折基线 `q1-robustness-cv-baselines-v1` 均为 accepted；独立
R2 在隔离目录复现两次，退出码均为 0，核心 JSON 逐字节一致，15 个指标差值为 0，四个响应的
特征矩阵均排除全部响应字段及 `sleep_hours` 派生列。审核结论为
`REPRODUCIBLE_WITH_WARNINGS`，无阻断 finding。

## Gate B：路线

三名不知道映射关系、未读取试点报告的独立评阅者均选择 R42（知识/工作台路线）。R17 分数为
64、66、61，平均 63.6667；R42 分数为 96、88、92，平均 92.0000，平均分差 28.3333。
3/3 选择 R42，且每名评阅者均确认 R42 数学正确性未下降，无 P0，因此 `gate_b_pass=true`。

## Gate C：论文

两份论文只使用相同的 v4 primary、同折 baseline、执行记录和图表。P17 分数为 48、55、55，
平均 52.6667；P42 分数为 86、75、88，平均 83.0000，平均分差 30.3333。3/3 选择 P42，
各评阅者的数学正确性维度均不低于 P17，无 P0，因此 `gate_c_pass=true`。

Gate C 证明的是同一实验事实经过工作台组织后，论证和结论边界显著改善，不代表模型样本外性能
提高。三份报告仍保留系数与区间、Poisson 过度离散替代模型、重采样稳定性和完整多问覆盖等 P1。
