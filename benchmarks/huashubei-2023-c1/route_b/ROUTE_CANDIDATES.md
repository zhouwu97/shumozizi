# 候选路线 B

## B1 共享协变量字典驱动的分响应证据链（推荐）

- 数学核心：构造统一的清洗、编码和分层对象，再按响应支持集选择 OLS、计数回归和多项
  Logistic；所有模型共享样本索引、变量字典和诊断口径。
- 共享数学对象：共享样本索引、基础协变量字典和固定验证折，并显式维护响应专用矩阵
  $X_b,X_s,X_w,X_m$；每个矩阵都排除当前目标、目标派生列和其余响应字段。
- 各问递进：先校准清洗与类别语义，再建立简单基线，随后比较主模型，最后用重采样与敏感性
  检验确定哪些关联可保留、哪些只能给出否定或不确定结论。
- 借鉴模式：来自葡萄酒统计评价卡的“共享样本对齐键、基线先行、自然子群比较、不能预测时
  给出否定结论”。
- 当前题改造：不迁移评委一致性、葡萄等级或原回归式；将“可信标签”改造成四类响应的编码
  与缺失规则核验，并明确观察性数据不能支持因果。
- 数据支持：390 个样本只允许低自由度主效应和少量预注册交互，不支持大规模黑箱搜索。
- baseline：均值预测、类别众数、无协变量 Poisson/Logistic；与主模型共享拆分和指标。
- 主模型：OLS（睡眠）、Poisson 并检查过度离散（睡醒）、多项 Logistic（行为和入睡方式）。
- 验证设计：五折分层交叉验证、Bootstrap 系数区间、VIF/残差/过度离散诊断、异常时间处理
  敏感性；训练指标与样本外指标分开报告。
- 论文主线：共同数据对象 -> 四类响应的公平对照 -> 稳定关联与不可预测边界 -> 非因果结论。
- 实现成本：中等且有界；每个响应一个 baseline 和一个 primary 实验族。
- 失败退路：Poisson 过度离散则切换负二项；类别稀疏则合并有题意依据的层级或只报告描述统计。

## B2 正则化多任务预测与可解释基线对照

- 数学核心：对不同响应分别使用正则化模型，通过统一外层交叉验证比较泛化，而不共享损失函数。
- 共享数学对象：固定的样本折、设计矩阵、缺失处理和评价指标注册表。
- 各问递进：基线 -> 正则化 GLM -> 非线性树模型仅作消融 -> 稳定性选择。
- 借鉴模式：迁移“基线公平、分层复核、样本外判据”，不迁移论文中的逐步回归和阈值。
- 当前题改造：重点判断预测增益是否足以支持复杂度，而不是堆叠算法名称。
- 数据支持：仅允许小网格和嵌套交叉验证，限制模型拟合次数。
- baseline：与 B1 相同。
- 主模型：Elastic Net、正则化 Logistic，树模型仅在样本外显著提升时保留。
- 验证设计：嵌套五折、校准、Bootstrap 选择频率、标签置换负对照。
- 论文主线：复杂度是否带来可复验增益；若没有则以负结果收束。
- 实现成本：高于 B1，但预算可控。
- 失败退路：回到 B1，并保留“复杂模型无明显增益”的消融证据。
