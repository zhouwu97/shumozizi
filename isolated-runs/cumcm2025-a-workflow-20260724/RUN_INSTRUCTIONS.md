# 2025 A 题隔离重跑约束

本目录是一次全新的 Capability-First v3 生产运行。求解任务只能读取 `problem-materials/` 中的原始题目材料，并只能在本目录内写入本次运行产物。

- 必须完整使用仓库内 `$mathmodel-workflow`，完成 Q1 至 Q5，不得跳过目标语义预审、能力路由、真实实验、科学红队、可视化、论文、PDF 盲审、机械 QA 和最终交付审核。
- 禁止读取仓库或机器上的同题旧运行、旧答案、旧论文、诊断报告、用户报告以及公开同题答案；禁止联网检索同题解答。
- Q5 必须把题面允许的全部动作数作为可变决策，完整挑战最多 15 个动作；只优化或复算 3 个动作不得通过。
- MATLAB/Octave 只有在当前运行真实探测可用后才能启用。启用后，高风险几何或优化结论必须由实际 `.m` 脚本生成证明/验证图，图进入论文正文并解释其支持的命题与不能证明的边界。
- 最终 PDF 的源码附录必须直接收录完整可运行的 Python/MATLAB 源码文本，不能只给路径、链接或文件清单。
- 目标语义预审、科学红队、PDF 盲审和最终交付审核必须使用四个互不相同的全新 Codex 任务。最终审核按数学建模竞赛标准自由严格判断，禁止按固定表格勾选代替结论。
- 审查后只允许一次集中修订；修订后必须新建独立审核任务复审，第二次仍不通过就停止并保留失败证据。
