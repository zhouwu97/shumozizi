# 建模 Cookbook

本目录只提供候选方法、适用条件和失效模式。使用时选择一个与题目数学本质匹配的文件，连同
`model-selection-matrix.md` 最多读取两个知识文件；不要全量加载或把模板视为答案。

来源和许可边界见仓库根目录的 `THIRD_PARTY_NOTICES.md`。

- `prediction.md`：预测、回归和时序；
- `optimization.md`：资源配置、调度和决策；
- `mechanistic.md`：机理建模与参数反演；
- `evaluation.md`：综合评价、排序与权重；
- `statistical-learning.md`：统计推断、分类与机器学习；
- `network-systems.md`：网络、路径、匹配与传播系统。
