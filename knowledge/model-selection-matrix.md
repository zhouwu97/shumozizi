# 模型选择矩阵（候选生成器）

来源：基于 `Lupynow/math-modeling-skills` 的模型选择思路独立整理。矩阵只用于生成候选；最终路线由题意、数据、约束和 probe 决定。

| 数学本质 | 优先候选 | 首个低成本检查 | 常见失效边界 |
| --- | --- | --- | --- |
| 时序预测 | 季节性基线、回归、状态空间、树模型 | 时间切分与持久性 baseline | 使用随机切分造成未来泄漏 |
| 静态回归 | 线性/正则化、树、层级模型 | 残差与简单线性 baseline | 小样本高维过拟合 |
| 分类识别 | 逻辑回归、树、概率模型 | 类别比例与 PR-AUC | 用准确率掩盖稀有正类 |
| 离散优化 | 整数规划、网络流、启发式 | 可行性与贪心 baseline | 目标正确但约束漏建 |
| 连续优化 | 非线性规划、动态规划、仿真优化 | 约束、尺度与多初值 | 局部最优或不可行 |
| 机理/反演 | 守恒方程、ODE/PDE、参数估计 | 合成真值恢复 | 参数不可辨识 |
| 综合评价 | 指标体系、熵权/主观权重、排序 | 指标方向和权重扰动 | 指标重复计量或排名脆弱 |
| 网络与调度 | 图模型、最短路、匹配、流 | 小规模枚举对照 | 忽略容量或时间依赖 |

至少比较一个替代路线，并写明为什么该题的数据、目标和时间预算让当前路线更合适。
