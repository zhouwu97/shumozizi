# 模型检验菜单（按主张选择）

来源：基于 `Lupynow/math-modeling-skills` 的验证菜单独立整理。不要为凑完整性机械执行全部项目。

- 预测：时间/主体隔离、基线、泛化误差、残差、外推边界、区间或情景不确定性。
- 分类：类别比例、宏 F1 或 PR-AUC、阈值、混淆矩阵、正类正控制、概率校准。
- 优化：可行率、目标值、简单规则 baseline、多初值或界、参数扰动、极端场景、时间复杂度。
- 机理/反演：合成数据恢复、已知真值、参数相关性、多初值、极限行为、误差传播。
- 评价/排序：权重来源、指标方向、权重扰动、排名翻转、替代方法、一致性区间。

每个检验必须回答“它能推翻哪一个主张”，不能改变决策的检验应缩减或省略。
