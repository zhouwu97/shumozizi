# Uppercase

# upper

Converts a string or content to uppercase.

## Example

```typst
#upper("abc") \
#upper[*my text*] \
#upper[ALREADY HIGH]
```

```typst
#upper(
  text
) -> str content
```

## Parameters

- text:
  - description: The text to convert to uppercase.
  - type: str | content
  - default: None
